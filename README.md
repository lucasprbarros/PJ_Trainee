Nós faremos uma análise dos dados climáticos da cidade de São Paulo. Começaremos com uma apresentação e análise do banco de dados, construindo histogramas e calculando alguns momentos estatísticos. Após isso, faremos um mapa de calor das covariâncias das variáveis; caso a covariância da temperatura e precipitação seja positiva e significativa, relizaremos uam regressão linear simples dessas duas variáveis. Ao final, faremos uma série temporal simples explorando as temperaturas de São Paulo nos últimos cinco anos, sobrepondo os índices de enchentes no gráfico, a fim de comparar e analisar se há alguma relação.

Banco de dados retirado do Instituto Nacional de Meteorologia (inmet)
